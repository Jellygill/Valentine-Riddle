Tailwind Config: Extended with 'Press Start 2P' font for the pixelated aesthetic.
Theme: Custom pink palette (love-100 to love-900) defined in CSS variables.
Assets: Using CSS-based pixel art or pixel-font icons where possible to maintain the 8-bit style without external assets.
Animations: Uses Framer Motion for complex sequences (floating hearts, dialog popups).
